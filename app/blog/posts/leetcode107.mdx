## 107. Binary Tree Level Order Traversal II

Given the `root` of a binary tree, return the bottom-up level order traversal of its nodes' values. 
(i.e., from left to right, level by level from leaf to root).


Example 1
`````
Input: root = [3,9,20,null,null,15,7]
Output: [[15,7],[9,20],[3]]
`````

Example 2
`````
Input: root = [1]
Output: [[1]]
`````

Example 3
`````
Input: root = []
Output: []
`````

Constraints:

- The number of nodes in the tree is in the range `[0, 2000]`.
- `-1000 <= Node.val <= 1000`

## Solution
Approach 1: Iteration
````JAVA
// Time complexity: O(n)
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrderBottom(TreeNode root) {

        List<List<Integer>> result = new ArrayList<>();
        if (root == null) { return result; }

        Deque<TreeNode> deque = new ArrayDeque<>();
        deque.offer(root);
        while (!deque.isEmpty()) {
            List<Integer> itemList = new ArrayList<>();
            int size = deque.size();
            
            for (int i = 0; i < size; i++) {
                TreeNode curr = deque.poll(); //removeFirst
                itemList.add(curr.val);

                if (curr.left != null) { deque.offer(curr.left); }
                if (curr.right != null) { deque.offer(curr.right); }
            }
            result.add(itemList);
        }
        Collections.reverse(result);
        return result;
    }
}
````

Approach 1.2: Iteration without reversing the result list
````JAVA
// Time complexity: O(n)
class Solution {
    public List<List<Integer>> levelOrderBottom(TreeNode root) {

        List<List<Integer>> result = new LinkedList<>();
        if (root == null) { return result; }

        Deque<TreeNode> deque = new ArrayDeque<>();
        deque.offer(root);
        while (!deque.isEmpty()) {
            List<Integer> itemList = new ArrayList<>();
            int size = deque.size();
            
            for (int i = 0; i < size; i++) {
                TreeNode curr = deque.poll(); //removeFirst
                itemList.add(curr.val);

                if (curr.left != null) { deque.offer(curr.left); }
                if (curr.right != null) { deque.offer(curr.right); }
            }
            result.addFirst(itemList);
        }
        return result;
    }
}
````
